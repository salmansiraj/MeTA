# MeTA

### Purpose
Creating a mobile application that serves as an integrated platform for MTA customers to keep track of and refill/renew their Metrocard balance and provide nearby transportation routes

### Technologies Used 
- Swift 
- RealmDB 
- Google Maps API 
- Square Payment API 
- CoreML
